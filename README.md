# NHAEUtilities

Standalone GTNH 1.7.10 mod scaffold for `NHAEUtilities`, built with RetroFuturaGradle in the same style as `AE2PatternGen`.

## Status

This repository currently contains only the blank build scaffold, mod metadata, and the minimal Forge entrypoint/proxy classes required to compile.

## Build

On Windows:

```powershell
.\gradlew.bat compileJava
```

On Unix-like systems:

```bash
./gradlew compileJava
```
