## v0.1 - Initial Release

### 样板生成器 / Pattern Generator
- 批量将 GregTech 机器配方编码为 AE2 处理样板
- 正则过滤 + 矿辞替换 + 电压等级过滤
- 显式过滤语法：`[ID]` / `(矿辞正则)` / `{显示名正则}`
- 配方冲突 GUI 手动解决
- 虚拟样板存储与等量消耗空白样板

### 样板路由 / Pattern Routing
- AE2 编码样板自动路由到 GT 多方块输入仓
- 空白仓自动配置与 ME 网络物品提取
- 配方映射分析 GUI（重复/单次类型分解、NEI 书签集成）
- `/nau repairrouting` 修复命令

### 超级无线设置工具 / Super Wireless Kit
- AE2 频道设备无线连接到 ME 控制器
- QUEUE/BIND 双模式 + 批量递归收集
- 虚拟 GridConnection 持久化绑定
- AE2 安全权限检查
